# RektDeveloper

Welcome to **RektDeveloper**, a blog and resource platform focused on insights, market analysis, and development trends, especially in the world of cryptocurrency and TypeScript.

This repository powers the RektDeveloper website, hosted via GitHub Pages.

---

## Features

- **Responsive Design**: Fully responsive across mobile and desktop.
- **Dynamic Content**: Blog posts dynamically fetched via JavaScript.
- **Modern UI**: Clean design with a vibrant color scheme, interactive icons, and an easy-to-navigate layout.

---

## View the Website

You can view the website live at:

## Getting Started

### Prerequisites

- A modern browser (Chrome, Firefox, etc.)
- Code editor (VS Code, Sublime, etc.)

### Installation

To view the project locally:

1. Clone this repository:

    ```bash
    git clone https://github.com/Rekt-Developer/rekt-developer.github.io.git
    ```

2. Open `index.html` in your browser to start viewing the website locally.

---

## Customizing the Website

### 1. Update the Hero Section
   Modify the content inside the `<section class="hero">` tag in `index.html` to reflect your brand's messaging.

### 2. Modify the Blog Section
   The blog section dynamically pulls content. You can update the blog posts API endpoint in the JavaScript or hard-code content for specific insights.

---

## Technologies Used

- **HTML5**: For the structure of the website.
- **CSS3**: For styling, including custom properties and responsive layouts.
- **JavaScript**: For dynamic content fetching and mobile navigation.
- **Font Awesome**: For icons and scalable vector graphics.
- **Google Fonts**: For custom fonts, including **Inter** and **Siyam Rupali** for Bangla support.
